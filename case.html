Case: Como utilizar o git em projetos

Utilizando o Manual do Git Maneirinho para esse site

Para demonstração resolvemos utilizar o git de fato para a realização desse site. Dessa forma conseguimos de forma colaborativa alterar arquivos em paralelo.

O git pode ser muito interessante para que num trabalho tanto em grupo, quanto individual, seja possível guardar todas as versões de um projeto, sem que seja necessário salvar n arquivos com nomes diferentes!
