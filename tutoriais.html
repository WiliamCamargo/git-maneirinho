Como criar sua conta no GitHub

1º Passo

Acesse o site do GitHub clicando aqui e clique em Sing Up no canto superior direito.

2º Passo

Preencha com as suas informações de:

  • E-mail;
  • Senha;
  • Apelido que irá aparecer na plataforma;
  • E se você quer receber e-mails sobre atualizações e anúncios;

3º Passo

Faça a verificação da sua conta para que o GitHub saiba que você é uma pessoa de verdade.

4º Passo

Depois de verificado, clique em Create account.

5º Passo

Selecione quantas pessoas irão trabalhar com você e se você é um estudante ou aluno.

6º Passo

Selecione os recursos que você está interessado em usar.

7º Passo (último)

Escolha se você quer utilizar a versão Free ou a versão Pro com benefícios adicionais para quem é estudante (esta versão é válida somente enquanto você é um estudante).
